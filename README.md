# Geospatial Dashboard Stack Library

A compact, searchable library of open geospatial dashboard tools, platform frameworks, standards, and data feeds.

The library distills research into a practical decision aid for building dashboards with MapLibre GL JS, deck.gl, Apache ECharts, Leaflet, OpenLayers, CesiumJS, kepler.gl, TerriaJS, MapStore2, GeoNode, Grafana Geomap, Superset, STAC, OGC APIs, PMTiles, COGs, TiTiler, and authoritative data feeds.

## What is inside

- `index.html` - a self-contained searchable frontend with 50 structured entries.
- `codex-skills/geospatial-dashboard-architect` - a Codex skill for selecting dashboard stacks and delivery bundles.
- `codex-skills/geospatial-data-feed-selector` - a Codex skill for selecting authoritative feeds and production data posture.

## Use

Open `index.html` in a browser. Search by tool, standard, feed, license, caveat, or use case. Filter by category, best fit, or license.

## Default Stack

For most greenfield custom dashboards:

- MapLibre GL JS for open vector maps.
- deck.gl for GPU analytical overlays.
- Apache ECharts for linked charts.
- STAC for earth-observation discovery.
- OGC API Features/Tiles, PMTiles, COGs, and TiTiler for durable data delivery.

Use TerriaJS, MapStore2, or GeoNode when catalog governance, metadata, publishing, permissions, and portal workflows matter more than total UI freedom.

## Expanded Source Coverage

The library includes additional high-quality sources for catalog discovery, hazards, climate, hydrology, protected areas, environmental data, biodiversity, bathymetry, population, humanitarian response, transit, addresses, and federal NGDA discovery.
